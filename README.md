# Projeto ODS 12 - Consumo e Produção Responsáveis

##  Descrição
Este projeto consiste em uma aplicação backend desenvolvida com Node.js e Express,
com integração a um banco de dados MySQL.

O sistema permite o gerenciamento de produtos sustentáveis, alinhado com a
ODS 12 da ONU, que promove o consumo e produção responsáveis.

## Objetivo
Permitir o cadastro, visualização, atualização e exclusão de produtos,
facilitando o controle de informações de fornecedores e incentivando práticas sustentáveis.

## Tecnologias Utilizadas
- Node.js
- Express
- MySQL
- HTML e JavaScript

## Como executar o projeto

1. Instalar dependências:
npm install
2. Criar database
executar o arquivo ods12_db.sql
4. Iniciar servidor:
node server.js
5. Acessar no navegador:
http://localhost:3000
